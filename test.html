FeedbackAgent SDK — Test Page
v1.0.0

Visual Feedback Widget

This page tests the FeedbackAgent SDK. Click the blue Feedback button in the bottom-right corner to trigger the screenshot and submission flow.

Note: The SDK sends feedback to http://localhost:8000/feedback. Open DevTools → Network to inspect the payload even if no backend is running.

Screenshot Capture

Uses html2canvas to capture the full page before the modal opens.

Clean Modal

Title, description, and screenshot preview with loading + success states.

Zero Dependencies

IIFE build bundles everything. No React required on the host page.

ESC to Close

Press Escape at any time to dismiss the modal without submitting.
